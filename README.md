## Getting Started

### 1. Setup environment variables

Create a `.env.local` file in the root of the project with the following content:

```env
AUTH_SECRET=

AUTH_GITHUB_ID=
AUTH_GITHUB_SECRET=

AUTH_GOOGLE_ID=
AUTH_GOOGLE_SECRET=
```

> ℹ️ Make sure to fill in the correct values for each field above.  

### 2. Run the development server

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
